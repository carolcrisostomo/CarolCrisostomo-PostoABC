# Posto ABC - Projeto com Fortes Report integrado

Este pacote foi gerado para RAD Studio 12 (Delphi). 
O formulário de relatório foi criado usando o Fortes Report (CE).

## Como usar
1. Abra `PostoABC.dpr` no RAD Studio 12.
2. Certifique-se de que o pacote Fortes Report CE está instalado no Delphi (componentes RL* disponíveis).
3. Crie o banco executando `db/script_criacao.sql` ou use seu .fdb. (Firebird 3.0)
